# 接口文档

## 用户模块
### 1.1 登录接口

``` http
POST /api/users/login
```

req
``` js
{
    account: "272798802",
    password: "123456"
}
```

res
``` js
{
    code: 200,
    msg: '登录成功',
    data: {
        token: '...'
    }
}
```
